# North Star Nanny Agency Website

A professional, modern website for North Star Nanny Agency - a premium nanny placement service in Utah County.

## Features

- **Multi-page Application**: Built with React Router for smooth navigation
- **Responsive Design**: Mobile-first approach works seamlessly across all devices
- **Professional Branding**: Soft neutral color palette (sage, cream, warm gray) with modern serif headings
- **Conversion-Focused**: Strategic CTAs and clear pricing throughout

## Pages

1. **Home** - Hero section, problem/solution, services overview, and social proof
2. **How It Works** - 5-step process timeline with detailed explanations
3. **Services & Pricing** - Transparent pricing cards and service comparisons
4. **Vetting Process** - 7-step screening process with trust badges
5. **About** - Founder story, values, and local expertise
6. **Resources** - Downloadable guides, FAQs, and local partners
7. **Contact** - Comprehensive intake form and contact information

## Design System

- **Colors**: Sage green primary (#7A9B8E), cream background (#FDFCFA), warm grays
- **Typography**: Playfair Display (serif headings), Inter (body text)
- **Components**: Rounded buttons, soft shadows, generous white space
- **Icons**: Lucide React for consistent iconography

## Technical Stack

- React 18
- React Router 7
- Tailwind CSS 4
- TypeScript
- Lucide React Icons
- Radix UI Components

## Development

The website is built with a focus on:
- Clean, maintainable code
- Reusable components
- SEO-friendly structure
- Fast loading times
- Accessible design
